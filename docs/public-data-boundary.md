# Public Data Boundary

This repository is a free public gallery created by PaywallPro.

It is designed to make a small part of PaywallPro's subscription research dataset available for learning, reference, and discovery.

## Included in this public repo

- Selected app case pages
- Representative paywall screenshots
- Basic pricing summaries
- Paywall pattern labels
- Selected monetization signals
- Category-level browsing
- Links to explore more in PaywallPro

## Not included in this public repo

- Full screenshot history
- Complete onboarding flows
- Version-by-version paywall changes
- Historical pricing tests
- Regional paywall differences
- Advanced revenue analysis
- Full app database
- Advanced filters and comparisons

## Why the boundary exists

The public gallery is designed to be useful on its own while keeping the full PaywallPro product valuable.

The repo helps users discover paywall examples and learn common monetization patterns.

PaywallPro provides the deeper workflow for teams that need complete research, historical tracking, and subscription strategy support.

## Recommended usage

You may use this repository for:

- Product research
- UX reference
- Pricing inspiration
- Internal team discussion
- Educational writing
- Public commentary with attribution

You may not use this repository to bulk copy, repackage, resell, or build a competing commercial paywall database.
