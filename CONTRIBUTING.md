# Contributing

This repository is generated from PaywallPro data. Manual edits to files under `apps/`, `screenshots/`, `categories/`, and `data/` may be overwritten by the generator.

Useful contributions:

- Report broken images or links.
- Suggest public data fields that would improve the gallery without exposing full PaywallPro history.
- Suggest better category indexes or README wording.

For complete paywall history and deeper analysis, use [PaywallPro](https://www.paywallpro.app/?utm_source=github&utm_medium=open_dataset&utm_campaign=paywall_gallery).
